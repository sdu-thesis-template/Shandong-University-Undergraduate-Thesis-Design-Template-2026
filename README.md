# 2026山东大学本科毕设模板-机电(低空)

本项目是基于 [Shandong-University-Undergraduate-Thesis-Design-Template](https://github.com/KYRIE-LI11/Shandong-University-Undergraduate-Thesis-Design-Template) 仓库适配整理的 2026 版山东大学本科毕业论文模板，并已在 CTex 环境下编译通过。

使用时直接下载本项目的 zip 文件并导入 Overleaf，在菜单中选择 XeLaTeX，TeX Live 版本选择 2024 或 2025 即可。
